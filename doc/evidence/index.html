Use correctly

Evidence rules

  • Link an artifact from a handbook page only when it proves a current claim.
  • Never convert a run report into current architecture by placing it in the primary navigation.
  • Preserve timestamps and content; add interpretation in a separate current page or ADR.
  • Do not relocate runtime-owned paths merely to make the documentation tree look tidier.
