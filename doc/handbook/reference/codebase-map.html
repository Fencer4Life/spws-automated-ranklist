Analysis tools

  • Use Graphify first for architecture, paths and impact: query, explain, affected, path.
  • Use language tooling/source inspection for exact symbol definitions, types and call sites.
  • Use repository search for literals, SQL objects, YAML and prose.
  • Refresh Graphify before committing architecture-affecting code.

Database navigation

Database contracts

Migrations are the authoritative evolution record, but current behavior is the result of the full ordered migration set. Search the latest definition of an object rather than assuming the first migration remains current. Never edit a deployed migration; add a corrective migration.

Views and RPCs are application contracts. When adding an event field, review every reconstructed view and round-trip function that owns the projection.
