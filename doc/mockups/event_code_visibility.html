Event-code visibility — 2 options

Goal: make PEW3, PEW7, IMEW, MSW visible on each event card so Telegram /promote and /complete commands can target the right one (e.g. Chania vs Salzburg).

Option A — Inline code chip (right of title)
A small monospace pill sits at the end of the title line, color-coordinated with the organizer band. Zero new rows, zero layout reflow on mobile (chip wraps to next line if cramped).
