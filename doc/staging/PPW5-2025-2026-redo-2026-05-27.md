# PPW5-2025-2026 — XML ingest re-do (2026-05-27)

**Event:** V Puchar Polski Weteranów, Gdańsk, 11 kwi 2026
**Cause of re-do:** same architectural divergence as PPW4 — ELIMINACJE
qualifier XMLs (`RESULTS_GRVETX{E,F,S}_*`) were being ingested as if they
were per-V-cat brackets, producing phantom 36-fencer V-cat rows. Plus a
new PPW5-specific bug exposed by the re-ingest: AltName uses `kat. N`
(Polish convention) instead of legacy `vN`, which the parser regex didn't
recognise → cross-V-cat draft duplication. Fixed in commit `347eb19`.

**Pre-wipe state (LOCAL):** 26 tournaments / 83 results, all 5 V-cats of
M-Epee carrying `int_participant_count = 36` (POL-only sum of the
47-fencer M-Epee eliminacje pool).

**Post-fix state (LOCAL):** 22 tournaments / 81 results.
ELIMINACJE files **skipped** (3): `RESULTS_GRVETXE_2026-0.xml`,
`RESULTS_GRVETXF_2026-1.xml`, `RESULTS_GRVETXS_2026-2.xml`. Per-V-cat /
joint-pool DE files **ingested** (18 files, producing 22 tournament rows
after V-cat splits).
